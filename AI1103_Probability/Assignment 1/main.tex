\section{Problem}
(Misc 6.23) Given that E and F are events such that \pr{E} = 0.6, \pr{F} = 0.3 and \pr{EF} = 0.2, find \pr{E|F} and \pr{F|E}\\

\section{Solution}
By definition, 
\begin{align}
\pr{A|B} = \frac{\pr{AB}}{\pr{B}}
\end{align}
Thus, we can write:

\begin{align}
\pr{E|F} = \frac{\pr{EF}}{\pr{F}} = \frac{0.2}{0.3} = \frac{2}{3}
\end{align}
In a similar manner:
\begin{align}
\pr{F|E} = \frac{\pr{EF}}{\pr{E}} = \frac{0.2}{0.6} = \frac{1}{3}
\end{align}
